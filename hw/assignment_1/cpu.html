The CPU

The CPU (central processing unit) is the "brain" of the computer. It processes the many requests given by programs. The faster the cpu, the faster you can gather, decode, and executes the program's instructions. If applying textures to a graphic in Photoshop takes long, then you might need a cpu upgrade. The 3 most important parts in the CPU are:

  • the arithmetic logic unit - handles the calculations of the cpu (arithmetic and logical operations).

  • the control unit - handles and directs all processor signals, makes every instruction run in the correct timing.

  • the cache - stores memory from frequently used programs and data so it can access them more quickly.
